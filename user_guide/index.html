QGML User Guide

This comprehensive user guide provides step-by-step tutorials and best practices for using the Quantum Geometric Machine Learning (QGML) framework.

Overview

The QGML framework enables:

  • Quantum-Enhanced Machine Learning: Leverage quantum geometric properties for classical data

  • Geometric Analysis: Analyze data manifolds using quantum information theory

  • Topological Insights: Discover topological invariants and phase transitions

  • Scalable Implementation: Efficient computation with PyTorch and JAX backends
